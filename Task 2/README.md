# Task 2 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset using statistical summaries and visualizations.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Visualizations
- Histogram
- Box Plot
- Pair Plot
- Correlation Heatmap
- Plotly Histogram

## Key Findings
- Most passengers were young adults between 20 and 30 years of age.
- The Age distribution is slightly right-skewed.
- A few elderly passengers appear as potential outliers.
- The correlation matrix shows relationships among numerical features.

## Conclusion
EDA helped in understanding the dataset, identifying patterns, detecting outliers, and exploring relationships between features before machine learning.
